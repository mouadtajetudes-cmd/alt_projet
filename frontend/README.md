# ALT Frontend

Frontend simple pour la plateforme ALT, construit avec Vite.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Fonctionnalités

- ✅ Test de tous les microservices via le gateway
- ✅ Connexion WebSocket temps réel
- ✅ Interface responsive
- ✅ Design moderne
